position

A Position is simply the name for an asset that a buyer actually has possession of. While Apple’s stock exists in the world as an asset, it only becomes a person’s position if they own Apple stock. A Position object as defined here has a symbol, quantity, and price associated with it.

class position.Position(alpaca_api, data_settings, symbol, initial_quantity)

This is a container for an individual position, like ETH or IVV. An instance is meant to be stored inside of a portfolio, so the position has a quantity currently held and a current price associated with it.
